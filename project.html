Tri-Sword Project

This project is an adventure RPG designed and programmed comepletly by me. I have programmed the engine completly from scratch and have been working on it for 3-4 months on and off between work. The Game consists of the main character and a world that he traverses. the story revolves around a sword split into 3 pieces that are scattered around this world. naturally, the main character has to find these tree pieces and defeat the darkness within the world. thats literally it.
